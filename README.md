# trading-bots

Monorepo for trading bot projects.

## Standards

Every project in this repo follows shared security, privacy, and abuse-control
standards. Before shipping anything:

- Read [`CLAUDE.md`](./CLAUDE.md) — the standards that apply to all projects
- Walk [`SECURITY.md`](./SECURITY.md) — the pre-launch checklist

These exist because trading bots handle money and credentials. A leaked
exchange API key or an unprotected endpoint can drain an account or burn an
API bill in minutes.
